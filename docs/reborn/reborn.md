## 기본 조작 및 시작 가이드
<br>

- 7세대까지 구현 ( All-gen 모드 사용 시 9세대 구현 )
- 다양한 필드 시스템 ( 이에 대한 자세한 설명은 인게임 내 **포켓기어 - 필드노트**를 참고 )
- 개체값과 노력치가 적용된 실전형 포켓몬 등장
- 레벨캡 존재 ( 소지한 배지에 따라 특정 레벨 이상으로 올라가면 명령을 따르지 않는 구간이 존재 )
- 패스워드 시스템
- 분기별 루트 존재 & 캐릭터들의 관계 포인트에 따른 엔딩씬 차이

- F1 키로 키 설정이 가능

※ 중요키 ※
- Z 키 - 달리기
- X 키 - 메뉴
- C 키 - 확인
- A 키 - 아이템 정렬 / 메가 진화 / Z 기술
- S 키 - 등록한 아이템 사용
- D 키 - 퀵세이브
- ALT 키 - 터보
- Ctrl + C 키 - 박스 다중선택
- Q / W - 페이지 넘기기 , 배틀 정보 확인
- F12 - 게임 재시작

## 야생 포켓몬 지닌물건
<br>
훔치기나 탐내기를 사용하면, 야생 포켓몬이 가진 아이템을 훔칠 수 있습니다.

기술을 사용하는 포켓몬이 이미 도구를 지니고 있으면 빼앗지 못합니다.

이 특성을 가진 포켓몬이 선두에 있으면, 아이템 출현 확률이 50%, 5%, 1%에서 60%, 20%, 5%로 증가합니다.

| 지닌물건 |  확률  |
|:--------|:-------|
| 달콤한꿀 | - 50% : 비구술<br>- 5% : 세꿀버리, 춤추새, 에블리, 에리본 |
| 달의돌 | - 5% : 삐삐, 픽시, 푸린, 푸크린, 삐, 루나톤 |
| 태양의돌 | - 5% : 솔록 |
| 고운깃털 | - 50% : 갈모매, 패리퍼 |
| 작은버섯 | - 50% : 파라스, 파라섹트, 버섯꼬, 버섯모, 깜놀버슬, 뽀록나, 자마슈, 마셰이드 |
| 큰버섯 | - 5% : 파라스, 파라섹트, 버섯꼬, 버섯모, 깜놀버슬, 뽀록나, 자마슈, 마셰이드 |
